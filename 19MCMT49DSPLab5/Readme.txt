#Problem Statement
Suppose you are visiting a mall and you have to bill your purchased items. There are 'n' counters available.

You have to choose a counter according to given below parameters:
- Check the counters for minimum number of people and join that queue.
- If all the counters have 0 people, then join any random queue.
- If all the counters have same number of people, then join any random queue.

For removing a person from queue you can choose any random queue provided it is not empty.
